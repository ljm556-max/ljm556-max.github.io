数组方法合集,包含ES6

2021-06-19- 2021-06-30

开发过程中，常用到Array数据类型，整理一下目前为止用过的数组方法，主要包括数组元素的添加、删除、遍历、查找等等……

数组添加、删除元素

1.1 arr.push()

  • 向数组得末尾添加一个或多个元素。返回新的长度。
  • 注意：返回得是长度不是数组。直接改变原数组。

1.2 arr.pop()

  • 从数组后面删除最后一个元素 ！只能删除一个
  • 返回删除的元素的值，改变原数组，如果数组已经为空，则pop()不改变数组，并返回underfined

1.3 arr.unshift()

  • 向数组得末尾添加一个或多个元素。返回新的长度。
  • 注意：返回得是长度不是数组。直接改变原数组。

1.4 arr.shift()

  • 删除数组的第一个元素， ！只能删除一个

  • 会改变原数组

1.5 arr.splice(i,n,….)

  • splice可用于删除、插入、替换数组元素

    • 删除—-返回被删除的元素，会改变数组
  • 插入—-返回插入元素之后，改变了的原数组
  • 替换—-返回元素被替换之后，改变了的原数组
  • 实质是在特定的位置删除元素之后再插入元素

数组的遍历

2.1 arr.forEach(cb)

  • forEach遍历数组是直接改变原数组，无return *即使有return，也不会返回任何值，并且会影响原来的数组*

  • callback的参数： value –当前索引的值

    　　　　　　　　　　index –索引

      　　　　array --原数组

2.2 arr.map(cb)

  • 映射数组(遍历数组),有return 返回一个新数组 。

  • callback的参数： value –当前索引的值

    　　　　　　　　　　index –索引

ps: arr.forEach()和arr.map()的区别

  • arr.forEach()是和for循环一样，是代替for。arr.map()是修改数组其中的数据，并返回新的数据。– arr.forEach() 没有return arr.map() 有return

遍历数组，查找特殊元素

3.1 arr.filter(callback)

  • 过滤数组，返回，满足条件的元素组成的新数组

3.2 **arr.every(callback) **

  • 依据判断条件，数组的元素是否全满足，若全满足则返回ture，其中有元素不满足就返回false。

3.3 **arr.some() **

  • 依据判断条件，数组的元素是否有一个满足，若有一个满足则返回ture

3.4 **arr.find(callback) **

  • 找到第一个符合条件的数组成员

3.5 arr.indexOf()

  • 查找某个元素的索引值，若有重复的，则返回第一个查到的索引值若不存在，则返回 -1
