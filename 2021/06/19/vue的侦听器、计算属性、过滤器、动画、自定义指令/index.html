2、计算属性

  • 计算属性含义： 本质还是属性，用来在模板中展示！

  • 计算属性其实和data的地位相同，都是vue实例的属性，都可以在模板中使用！

  • 出现的原因：一般在模板语法内使用表达式非常方便，模板也只是用于简单的运算，当表达式过于复杂时，在模板中放入太多逻辑会让模板过重且难以维护。为此，Vue提供了计算属性computed。

  • 如何使用：

  • 注意点：

    • 计算属性不要去修改！！！！
    • computed和methods的区别
      • computed 依赖于缓存！ 依赖的数据变量没有变化，他是不会重新执行！
      • methods 要每次都执行！
    • computed 和 watch的区别
      • computed 是为了简化模板里面的操作，对数据处理后进行展示； watch目的是监听某个变量的变化，执行响应的逻辑处理。

  • 真的修改计算属性

4、vue动画

  • 含义：Vue 在插入、更新或者移除 DOM 时，提供多种不同方式的应用过渡效果。

  • 何时使用：

    • 条件渲染 (使用 v-if)
    • 条件展示 (使用 v-show)
    • 动态组件
    • 组件根节点

  • 动画状态

    • 进入状态
      • 进入前 动画名-enter
      • 进入中 动画名-enter-active
      • 进入后 动画名-enter-to
    • 离开状态
      • 离开前 动画名-leave
      • 离开中 动画名-leave-active
      • 离开后 动画名-leave-to

  • 使用：

  • 代码

  • 自定义过渡类名

    • enter-class
    • enter-active-class
    • enter-to-class (2.1.8+)
    • leave-class
    • leave-active-class
    • leave-to-class (2.1.8+)

    他们的优先级高于普通的类名，这对于 Vue 的过渡系统和其他第三方 CSS 动画库，如 Animate.css 结合使用十分有用。
